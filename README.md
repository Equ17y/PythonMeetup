# Python meetup bot

## Установка и запуск
```
   git clone <URL>
   cd <directory>
```
1. Создайте виртуальное окружение:
```
2. python -m venv .venv
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Создать ```.env``` файл:
```
TELEGRAM_BOT_TOKEN=ваш_токен
```
4. Запуск бота:
```
python bot.py
```
